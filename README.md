# ИБ-Эффект
ИБ-Эффект (Calculatrade) - Это Веб-приложение для оценки рисков информационной безопасности, расчета остаточных рисков после внедрения защитных мер и анализа экономической эффективности (ROI/ROSI) инвестиций в ИБ.

# Запуск
- Склонировать репозиторий.
- Откройте файл index.html в любом современном браузере.
- Установка сервера не требуется (приложение работает на стороне клиента).

# Расчеты
Расчеты базируются на методологии ГОСТ Р ИСО/МЭК 27000-2021 "Информационная технология. Методы и средства обеспечения безопасности. Менеджмент риска информационной безопасности" и стандартных формулах управления рисками.

## Основная формула расчёта риска

Базовая оценка риска производится по формуле:
`Риск = Ущерб × Вероятность × Приоритет актива`

Эта формула выбирается в разделе «Критерии → Формула риска».
-  Все три параметра оцениваются в баллах (по умолчанию шкала 1–4).
-  Максимальное значение риска = 4 × 4 × 4 = 64 балла.

## Уровни риска и Риск-аппетит

| Уровень риска | Диапазон баллов |
|:-------------:|:---------------:|
|     Низкий    |      1 – X      |
|    Средний    |    X + 1 – Y    |
|    Высокий    |    Y + 1 – 64   |

Риск-аппетит — это максимально допустимый балл риска (например, ≤ 12 баллов).
- Все риски, превышающие значение риск-аппетита, считаются неприемлемыми и требуют обработки (внедрения защитных мер).

## Пересчёт риска после внедрения защитной меры

После применения защитной меры пользователь задаёт два параметра эффективности меры*:
- Снижение ущерба (в %)
- Снижение вероятности (в %)

Формула остаточного риска:
`Остаточный риск = (Ущерб × (1 − Снижение ущерба / 100)) × 
(Вероятность × (1 − Снижение вероятности / 100)) × Приоритет`

*Примечание: Если защитная мера полностью убирает вероятность (100%), то вероятность = 0 → риск = 0.*

## Экономическая эффективность защитной меры
Для перевода баллов риска в денежный эквивалент используется параметр «Стоимость 1 балла риска» (настраивается в критериях, по умолчанию 50 000 ₽).

Формула ожидаемого ущерба (в рублях):
`Ожидаемый ущерб = Баллы риска × Стоимость 1 балла риска`

### Net ROSI (Чистый возврат инвестиций в ИБ)
`Net ROSI = ((Снижение потерь − Стоимость меры) / Стоимость меры) × 100%`

### Чистый экономический эффект
`Чистый эффект = (Ожидаемые потери до мер − Ожидаемые потери после мер) − Стоимость всех мер`

### Экономический эффект (без вычета затрат)
`Экономический эффект = Сумма снижения ожидаемых потерь по всем рискам`
